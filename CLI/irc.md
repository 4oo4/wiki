## `irc`
View and manage irc connections

### Actions
| action | description |
| --- | --- |
| `status` | Shows status for specific irc connection |
| `restart` | Restart an irc connection |
| `stop` | Stops an irc connection |


### Examples
```bash
#shows the status for a specific irc connection
flexget irc status
```
