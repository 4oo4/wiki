## `templates`
View file templates, to be used with [notifer](/Plugins/Notifiers) plugins

### Actions
|  Option | Description |
| --- | --- |
|*optional:*<br>`--name <TEMPLATE NAME>`| Filter by template name
* Supports [table-styles](/CLI/--table-styles)


### Examples
```bash
# Shows all templates:
flexget templates 
# Filters by name 'rss':
flexget templates --name rss
```
