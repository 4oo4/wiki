# Torrentz
{{> Includes/PluginRemovedArchived }}

Search plugin which gives results from [torrentz.eu](http://torrentz.eu)

### Config
Takes a single option, the reputation of the results. Valid choices are `any`, `low`, `good`, and `verified`

**Example:**
```
torrentz: verified
```
