# Metainfo Series

### Syntax

```yaml
metainfo_series: yes
```

This will add the following fields to an entry:

* id (eg: `Cool Series S01E01`)
* proper
* proper_count
* release_group
* series_episode
* series_episodes
* series_guessed
* series_id
* series_id_type
* series_name
* series_parser
* series_season
